# E-Commerce Sales Analysis

## Project Overview
This project analyzes the Olist Brazilian E-Commerce dataset to identify sales trends, top-performing categories, regional performance, and customer review patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI
- Google Colab

## Key Findings
1. Beauty & Health generated the highest revenue.
2. November 2017 was the peak sales month.
3. São Paulo contributed the highest revenue.
4. Most customers gave 5-star reviews.
5. Revenue is concentrated in a few categories.
